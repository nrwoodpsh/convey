# ref/glossary — 용어집 (입력 정본, 상시 참조)

도메인 용어를 정의해 **AI·사람이 같은 말을 같은 뜻으로** 쓰게 한다. AI의 용어 오용(결재/결제, member/user, 취소/반품 등)을 막는 저비용·고효과 장치.

> 분류(##)는 도메인 순(research→content→컴포넌트→publishing→크로스커팅). 용어 추가 시 인덱스도 갱신.

## 작성 방식

도메인이 많으면 도메인별로 분리: `glossary/{NN}.{domain}.md`. 적으면 `glossary/01.terms.md` 하나로.

```markdown
| 용어 | 영문/코드 | 정의 | 혼동 주의 |
|:---|:---|:---|:---|
| 결재 | approval | 문서·요청의 승인 절차 | ≠ 결제(payment, 돈 지불) |
| 회원 | user | 로그인 계정 보유자 | ≠ 고객(customer, 구매 이력 기준) |
```

## 원칙

- 코드 식별자(영문)와 자연어(한글)를 **함께** 적어 매핑을 고정한다.
- 새 용어가 논쟁이 되면 여기서 확정하고, 코드·문서가 이를 따른다.
